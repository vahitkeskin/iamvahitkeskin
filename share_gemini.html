✨ Gemini AI (Google AI)

Google'ın en yetenekli çok modlu (multimodal) yapay zeka modeli ailesi.

Gemini 1.5 Pro Multimodal
Developer Portal Google AI Studio API Cookbook

Genel Bakış

Gemini, Google DeepMind tarafından sıfırdan yerel çok modlu (natively multimodal) olarak eğitilmiş en gelişmiş yapay zeka model ailesidir. Diğer modellerin aksine, görüntü veya sesi anlamak için ayrı yardımcı modeller kullanmak yerine, tüm bu veri türlerini tek bir nöral ağ üzerinde aynı anda işleyebilme yeteneğine sahiptir.

Model Mimarisi ve Ekosistem

  • Bağlam Penceresi (Context Window): Gemini 1.5 Pro, 1 milyondan fazla tokenı işleyebilir. Bu, 1 saatlik videoyu veya 700.000 kelimelik bir dokümanı tek seferde analiz edebileceği anlamına gelir.
  • Karmaşık Akıl Yürütme: "Chain-of-thought" ve mantıksal çıkarım yetenekleri, özellikle matematiksel problemler ve karmaşık kod mimarilerinde hata ayıklama süreçlerinde üstünlük sağlar.
  • Geliştirici Araçları: Vertex AI ve Google AI Studio üzerinden API erişimi sunarak, özel `system_instruction` parametreleri ile modele spesifik kimlikler ve görev tanımları atanabilir.

Geliştirici Araçları ve SDK Kurulumu

Projelerinize Gemini API'yi dahil etmek için şu komutları kullanabilirsiniz:

Python SDK
pip install -q -U google-generativeai
Node.js SDK
npm install @google/generative-ai

Uygulama Örnekleri (Python & Android)

Basit bir içerik oluşturma (content generation) örneği:

Python API Kullanımı
import google.generativeai as genai genai.configure(api_key="YOUR_API_KEY") model = genai.GenerativeModel("gemini-1.5-flash") response = model.generate_content("Explain AI in 3 points.") print(response.text)
