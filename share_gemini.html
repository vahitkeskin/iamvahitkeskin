✨ Gemini AI (Google AI)

Google'ın en yetenekli çok modlu (multimodal) yapay zeka modeli ailesi.

Gemini 1.5 Pro Multimodal
Developer Portal Google AI Studio API Cookbook

Yapay Zekada Yeni Dönem

Gemini, Google tarafından sıfırdan "çok modlu" (natively multimodal) olarak eğitilmiş, metin, kod, ses, görüntü ve videoyu aynı anda anlayıp işleyebilen bir modeldir. 1 Milyon+ token kapasitesine sahip bağlam penceresi (context window) ile devasa veri setlerini tek seferde analiz edebilir.

Geliştirici Araçları ve SDK Kurulumu

Projelerinize Gemini API'yi dahil etmek için şu komutları kullanabilirsiniz:

Python SDK
pip install -q -U google-generativeai
Node.js SDK
npm install @google/generative-ai

Uygulama Örnekleri (Python & Android)

Basit bir içerik oluşturma (content generation) örneği:

Python API Kullanımı
import google.generativeai as genai genai.configure(api_key="YOUR_API_KEY") model = genai.GenerativeModel("gemini-1.5-flash") response = model.generate_content("Explain AI in 3 points.") print(response.text)

Temel Yetenekler

Özellik Açıklama
Vision (Görüntü İşleme) Resimleri analiz ederek nesneleri tanımlar, metinleri okur veya betimler.
Code Execution (Kod Çalıştırma) Yazdığı kodları izole bir ortamda çalıştırıp sonucunu dönebilir.
Function Calling Kendi fonksiyonlarınızı API üzerinden çağırarak verilere bağlayabilirsiniz.
Structured Output Yanıtları doğrudan JSON formatında alarak API entegrasyonu sağlar.
Long Context 1 saatlik video, 30.000 satırlık kod veya 700.000 kelimelik metni analiz eder.

Nasıl Başlanır?

Geliştirme sürecinizi hızlandırmak için şu adımları izleyin:

  • Google AI Studio: Tarayıcı üzerinden prompt mühendisliği yaparak modelleri test edin ve API anahtarınızı alın.
  • Model Seçimi: Hızlı yanıtlar için "Gemini 1.5 Flash", derin analizler için "Gemini 1.5 Pro" kullanın.
  • Prompting: Çok modlu yeteneklerden faydalanmak için resim ve metni bir arada gönderen promptlar hazırlayın.

Uzman İpucu

Android geliştirmede Gemini modelini Jetpack Compose ile entegre ederek, uygulama içinde gerçek zamanlı asistanlar veya akıllı görüntü analiz süreçleri oluşturabilirsiniz.
