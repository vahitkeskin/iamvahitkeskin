📱 scrcpy (Screen Copy)

Android cihazınızı USB veya TCP/IP üzerinden mirroring ve kontrol etme aracı.

v2.4+ Open Source
GitHub Repository Latest Releases

Genel Bakış

scrcpy, Android cihazları bilgisayar üzerinden kontrol etmek için geliştirilmiş, yüksek performanslı ve düşük gecikmeli bir mirroring aracıdır. Root gerektirmez ve cihazın ekranını bilgisayarda gösterirken aynı zamanda fare ve klavye ile tam kontrol sağlar.

Kurulum Komutları

Her işletim sistemi için terminal üzerinden hızlı kurulum yapabilirsiniz:

Windows (WinGet)
winget install Genymobile.scrcpy
macOS (Homebrew)
brew install scrcpy
Linux (Debian/Ubuntu)
sudo apt install scrcpy

Gelişmiş Kullanım Seçenekleri

En sık kullanılan parametreler ve amaçları:

Ekran Kaydı ve Optimizasyon
scrcpy --record file.mp4 --max-size 1024 --bit-rate 8M --no-audio
  • --record: Ekranı .mp4 formatında kaydeder.
  • --max-size: Ekran genişliğini veya yüksekliğini sınırlar (FPS artışı sağlar).
  • --bit-rate: Video akış kalitesini belirler (varsayılan 8M).
  • --no-audio: Ses aktarımını kapatır (eski cihazlar için performans artırır).

Kısayollar (MOD = Sol Alt / Windows Tuşu)

İşlem Kısayol
Tam Ekran Modu MOD + F
Ana Sayfa (Home) MOD + H
Geri (Back) MOD + B
Son Uygulamalar MOD + M
Cihazın Ekranını Kapat (PC'de Aktif Tut) MOD + O
Pano İçeriğini Yapıştır MOD + V

Kablosuz Bağlantı Rehberi

Cihazınızı kablodan kurtarmak için şu adımları izleyin:

Wi-Fi TCP/IP Modu
adb tcpip 5555
adb connect [CİHAZ_IP]:5555
scrcpy

Not: Bu işlem için cihaz ve bilgisayar aynı Wi-Fi ağında olmalıdır.
